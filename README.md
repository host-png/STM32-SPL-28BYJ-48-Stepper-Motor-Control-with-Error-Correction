# STM32-SPL-28BYJ-48-Stepper-Motor-Control-with-Error-Correction
该仓库上传了一个我自己写的关于28BYJ-48步进电机的驱动，以及误差修正，使得其误差保持在0.0879度以内，而且就算以每步0.01度走，其误差也是在0.0879以内，当然前提是总角度要大于0.0879。This repository provides a 28BYJ-48 stepper motor driver with error correction.  The positioning error is kept within 0.0879°, even at a step size of 0.01°, as long as the total rotation exceeds 0.0879°.
